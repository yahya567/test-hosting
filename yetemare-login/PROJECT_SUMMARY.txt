================================================================================
                              OVERVIEW
================================================================================

A professional, computer-themed login page for Yetemare, an online learning 
platform. Built with pure HTML, CSS, and JavaScript. Deployed with Apache2 
in Docker Compose.

KEY FEATURES:
✓ Professional Tech Design (Deep Slate Blue & Teal)
✓ Multilingual Support (English, Amharic, Oromo, Tigrinya)
✓ Login with Phone Number & Password
✓ Password Reset Functionality
✓ Terms of Service Page
✓ Responsive Design (Mobile, Tablet, Desktop)
✓ Zero Dependencies (Pure HTML/CSS/JS)
✓ Docker Ready (Apache2 + Compose)

  - Method: GET
  - Parameters: msisdn, password, shortcode
  - Response: { status: "success|error", message: "..." }

PASSWORD RESET ENDPOINT:
  - URI: /reset
  - Method: GET
  - Parameters: msisdn, shortcode
  - Response: { status: "success|error", message: "..." }

SHORTCODE: 9643

================================================================================
                        USER FLOWS
================================================================================

LOGIN FLOW:
  1. User enters phone number and password
  2. Form validates input
  3. Request sent to backend API
  4. Backend returns success/error message
  5. Success: User redirected to dashboard
  6. Error: Message displayed, user can retry

PASSWORD RESET FLOW:
  1. User clicks "Forgot Password?"
  2. Form switches to reset mode
  3. User enters phone number
  4. Request sent to backend API
  5. Backend sends password via SMS
  6. Success message displayed
  7. Auto-redirect to login page after 2 seconds

TERMS OF SERVICE FLOW:
  1. User clicks "Terms of Service" link
  2. Dedicated page loads with full terms
  3. User can read all 10 sections
  4. Back button returns to login page

LANGUAGE SWITCHING:
  1. User clicks language dropdown
  2. Selects desired language
  3. All UI text updates immediately
  4. Language preference saved
  5. Persists across page reloads

================================================================================
                      CUSTOMIZATION GUIDE
================================================================================

CHANGE COLORS:
  Edit: html/css/style.css
  Update: :root CSS variables

ADD LANGUAGE:
  1. Edit: html/js/translations.js
  2. Add new language object
  3. Add option to selector in html/index.html

CHANGE API ENDPOINT:
  Edit: html/js/api.js
  Update: API_HOST and SHORTCODE constants

MODIFY STYLING:
  Edit: html/css/style.css or html/css/terms.css
  No build process required - changes apply immediately

                         GETTING STARTED
================================================================================

OPTION 1: Docker Compose (Recommended)
  $ docker-compose up -d
  $ open http://localhost

OPTION 2: Manual HTTP Server
  $ cd html
  $ python3 -m http.server 8000
  $ open http://localhost:8000

OPTION 3: Docker Manual Build
  $ docker build -t yetemare-login .
  $ docker run -p 80:80 yetemare-login
